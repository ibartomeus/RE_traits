# RE_traits
Code and data to reproduce a paper about response and effect traits.

You can find the paper as a preprint here: http://biorxiv.org/content/early/2016/08/29/072132

"Pollinator species traits do not predict either response to agricultural intensification or functional contribution."
Ignasi Bartomeus, Daniel Cariveau, Tina Harrison and Rachael Winfree.
doi: http://dx.doi.org/10.1101/072132
